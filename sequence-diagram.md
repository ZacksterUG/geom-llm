# Диаграмма последовательности: Основной бизнес-процесс

## Описание

Приложение для проверки геометрических доказательств (стереометрия) с 3D-визуализацией и ИИ-валидацией.

## Диаграмма

```mermaid
sequenceDiagram
    participant U as Ученик
    participant A as Приложение
    participant S as Сервер
    participant I as ИИ

    U->>A: Открывает приложение
    A->>S: Загрузить список задач
    S-->>A: Список задач
    A-->>U: Показать задачи

    U->>A: Выбрать задачу
    A->>S: Получить данные задачи
    S-->>A: Условие + фигура
    A-->>U: Показать условие и 3D-фигуру

    U->>A: Работать с фигурой
    U->>A: Ввести доказательство
    U->>A: Нажать "Проверить"

    A->>S: Отправить на проверку
    S->>I: Анализ доказательства
    I-->>S: Результат
    S-->>A: Ответ ИИ
    A-->>U: Результат + подсказка

    alt Если ошибка
        U->>A: Исправить и проверить снова
    end
```

## Этапы процесса

1. **Выбор задачи** — ученик видит список задач с фильтрами
2. **Работа** — интерактивная 3D-фигура + ввод доказательства
3. **Проверка** — ИИ анализирует каждый шаг
4. **Обратная связь** — результат + наводящая подсказка
5. **Итерация** — повтор до правильного решения
