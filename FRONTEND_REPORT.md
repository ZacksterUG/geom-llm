# Отчет по фронтенду Stereometry Proof Checker

## Выполнено

### Структура фронтенда
- **Стек**: Vite + React + TypeScript + Mantine + JSXGraph
- **Роутинг**: React Router (2 страницы)
- **Сборка**: успешна, Docker-образ готов

### Страницы
1. `/` — **Список задач** (TaskList)
   - Загружает задачи с `/api/tasks`
   - Карточки с названием, условием и кнопкой "Открыть"

2. `/task/:id` — **Страница задачи** (TaskView)
   - Левая колонка (1/3): условие, текст доказательства, кнопка "Проверить"
   - Правая колонка (2/3): интерактивная 3D сцена (GeometryCanvas)

### Компонент GeometryCanvas
- Использует JSXGraph с аксонометрической проекцией
- Режимы: "Точка" (добавление вершин), "Отрезок" (соединение точек)
- Сохраняет состояние фигуры в JSON

### Интеграция
- Прокси в Vite: `/api` → `http://localhost:8000`
- В docker-compose фронтенд работает через nginx на порту 3000
- API эндпоинты: `/api/tasks`, `/api/tasks/:id`, `/api/validate`

### Бэкенд
- FastAPI с поддержкой БД (PostgreSQL)
- Клиент Ollama для валидации доказательств
- Тестовые данные в БД

## Доступ
- **Фронтенд**: http://localhost:3000
- **Бэкенд API**: http://localhost:8000/docs
- **PgAdmin**: http://localhost:5050
